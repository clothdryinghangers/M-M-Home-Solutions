# Hero Slider Images

Add your hero slider images to this folder with the following names:

1. **ceiling-cloth-drying-hangers.jpg** - For Premium Ceiling Cloth Drying Hangers slide
2. **wall-mounted-cloth-hangers.jpg** - For Space-Saving Wall Mounted Hangers slide
3. **invisible-grills.jpg** - For Invisible Grills slide
4. **pigeon-safety-nets.jpg** - For Pigeon Safety Nets slide
5. **bamboo-roller-blinds.jpg** - For Elegant Bamboo & Roller Blinds slide
6. **shoe-racks.jpg** - For Premium Shoe Racks slide
7. **anti-bird-spikes.jpg** - For Anti Bird Spikes slide
8. **mosquito-nets.jpg** - For Mosquito Nets slide

## Image Requirements:
- **Format**: JPG or PNG
- **Recommended Size**: 1200x600px (16:9 aspect ratio)
- **File Size**: Under 500KB per image for optimal loading speed

## How to Add Images:
1. Save your images with the exact names listed above
2. Place them in this folder: `public/images/hero/`
3. The images will automatically appear in the hero slider
